 29:  * Title:        SHAREDFMLA (0x04BC) SharedFormulaRecord
 30:  * Description:  Primarily used as an excel optimization so that multiple similar formulas
 31:  *               are not written out too many times.  We should recognize this record and
 32:  *               serialize as is since this is used when reading templates.
 33:  * <p>
 34:  * Note: the documentation says that the SID is BC where biffviewer reports 4BC.  The hex dump shows

 37:  * @author Danny Mui at apache dot org
 38:  */
