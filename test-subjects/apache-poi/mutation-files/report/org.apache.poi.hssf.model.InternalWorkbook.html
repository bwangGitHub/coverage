727:     private void checkSheets(int sheetnum) {

757:             boundsheets.remove(sheetIndex);

762:         //  part point at deleted sheets. It's also ok to
763:         //  have the ExternSheetNumber point at deleted
764:         //  sheets.
765:         // However, the sheet index must be adjusted, or
766:         //  excel will break. (Sheet index is either 0 for
767:         //  global, or 1 based index to sheet)
