 20: /**
 21:  * A List of short's; as full an implementation of the java.util.List
 22:  * interface as possible, with an eye toward minimal creation of
 23:  * objects
 24:  *
 25:  * the mimicry of List is as follows:
 26:  * <ul>
 27:  * <li> if possible, operations designated 'optional' in the List
 28:  *      interface are attempted
 29:  * <li> wherever the List interface refers to an Object, substitute
 30:  *      short
 31:  * <li> wherever the List interface refers to a Collection or List,
 32:  *      substitute ShortList
 33:  * </ul>
 34:  *
 35:  * the mimicry is not perfect, however:
 36:  * <ul>
 37:  * <li> operations involving Iterators or ListIterators are not
 38:  *      supported
 39:  * <li> remove(Object) becomes removeValue to distinguish it from
 40:  *      remove(short index)
 41:  * <li> subList is not supported
 42:  * </ul>
 43:  *
 44:  * @author Marc Johnson
 45:  */
